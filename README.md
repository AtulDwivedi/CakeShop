# CakeShop
Welcome to **CakeShop**, a purely open source project base on JEE technologies.

## Dependencies
- h2 JAR file needs to be put in **WebContent/WEB-INF/lib** folder
- mail JAR file needs to be put in **WebContent/WEB-INF/lib** folder
